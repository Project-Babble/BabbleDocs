In order to use the Vive Pro Eye headset with Baballonia, you must download and use [Revision](https://github.com/Blue-Doggo/ReVision).

Once you have run the application, put `http://127.0.0.1:4444/left` and `http://127.0.0.1:4444/right` into your left and right eye cameras.

Please refer to their documentation page for installation, configuration and usage.
