# Resonite

Babble has 3 Resonite integrations. For all integrations, you must pass the `-ForceBabble` launch argument for things to work. We suggest the following in descending order:
1) *Mod-based* with [hantabaru1014's ResoniteBaballoniaEyeTracking](https://github.com/hantabaru1014/ResoniteBaballoniaEyeTracking)

:::info
We suggest using this!
:::

2) *Mod-based* with [hazre's VRCFTReceiver](https://github.com/hazre/VRCFTReceiver)

:::note
For [hazre's VRCFTReceiver](https://github.com/hazre/VRCFTReceiver) you *must* use version 5.1.1 of VRCFaceTracking and [the v2.0.5 Project Babble module](https://github.com/dfgHiatus/VRCFT-Babble/releases/tag/v2.0.5). You cannot use newer versions!
:::

:::warning
[The v2.0.5 Project Babble module](https://github.com/dfgHiatus/VRCFT-Babble/releases/tag/v2.0.5) may appear as v2.0.4. You can safely ignore this.
:::

3) *Native* with the [legacy Python App](/docs/developers/sourceCode/python.mdx)

:::danger
The Native integration only supports lower-face expressions!
:::
