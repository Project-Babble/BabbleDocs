Here are some cool open source projects that use Project Babble!

- [FoxyFace](https://github.com/Jeka8833/FoxyFace) - An application that allows you to control your avatar's face in VRChat using your real face (in desktop mode)
- [DockeredBabble](https://github.com/MagicBOTAlex/DockeredBabble) - Run Babble in a docker container
- [babble-rs](https://github.com/benaclejames/babble-rs) Babble, but in rust
- [ComboFT](https://github.com/MagicBOTAlex/ComboFT) An all-in-one eye tracking (ETVR) and face tracking (Babble) program
- [ProjectBabble-Backend](https://github.com/MagicBOTAlex/ProjectBabble-backend/tree/master]=) A self-hosted REST API for Babble
